# 📊 Dashboard Power BI — Analyse de données

## Description
Croisement de datasets, modélisation en tables faits/dimensions, 
mesures DAX et rapport visuel multi-pages avec analyse des tendances.

## Outils utilisés
- Power BI / DAX
- Excel / CSV (sources de données)

## Fonctionnalités
- Modélisation en étoile (tables faits/dimensions)
- Mesures DAX personnalisées
- Rapport multi-pages interactif
- Analyse des tendances
